# Tamagotchi Mini Project

## Описание

Это Android-приложение на Java, представляющее собой эмулятор тамагочи.  
Пользователь ухаживает за существом: кормит его, укладывает спать и играет с ним.  
Со временем состояние существа ухудшается, а задача игрока — продержаться как можно дольше.

## Возможности

- Изображение существа меняется в зависимости от его состояния.
- Есть параметры:
  - голод;
  - усталость;
  - скука;
  - счастье.
- Кнопки действий:
  - накормить;
  - спать;
  - играть;
  - перезапуск игры.
- Состояние существа постепенно изменяется со временем.
- При критических значениях существо умирает.
- Со временем игра ускоряется.
- Сохраняется лучшее время жизни.

## Технологии

- Java
- Android Studio
- SharedPreferences
- Material Design

## Структура проекта

- `MainActivity.java` — основная логика игры.
- `Tamagotchi.java` — модель состояния существа.
- `activity_main.xml` — интерфейс приложения.
- `drawable/` — изображения существа.
- `bg_gradient.xml` — фон приложения.

## Как запустить

1. Скачать или клонировать проект.
2. Открыть папку проекта в Android Studio.
3. Дождаться синхронизации Gradle.
4. Запустить приложение на эмуляторе или телефоне.
